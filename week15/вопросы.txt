4. Можно ли пропускать части for? Что получится, если написать for(;;)? 
Можно, но точку с запятой надо оставить. Цикл состояит из трех выражений - начальное значения, проверка условия и шаг. Если пропустить условие то цикл станет бесконечным, поэтому for(;;) это бесконечный цикл.

5. Самостоятельно разберитесь, как работает цикл while и приведите два примера кода с его использованием.
Цикл while работает таким образом, что код из тела цикла выполняется, пока условие в скобках истинно.
например
let i = 0;
while (i < 3) {
  alert( i );
  i++;
}

let i = 1, n = 5;
while (i <= n) {
    console.log(i);
    i += 1;
}
